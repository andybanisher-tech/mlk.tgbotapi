Модуль интеграции Telegram Bot API (mlk.tgbotapi) v3.3.0
========================================================
Партнёр: mlk (https://www.mirlk.ru)

Описание:
---------
Модуль предоставляет REST API для получения рекламных баннеров из нескольких инфоблоков на основе групп контрагента, хранящихся в Highload-блоке.

Что нового в версии 3.3.0:
--------------------------
- Добавлена возможность задавать шаблон ссылки для каждого инфоблока с плейсхолдером {value}.
- В настройках модуля добавлен предпросмотр сгенерированной ссылки.
- Улучшена гибкость настройки ссылок в ответе API.

Установка:
----------
1. Загрузите и установите модуль через Marketplace.
2. Перейдите в настройки модуля (Настройки → Настройки продукта → Список модулей → mlk.tgbotapi).
3. Настройте Highload-блок и добавьте инфоблоки.
4. Для каждого инфоблока укажите:
   - Свойство с группами.
   - Поля для имени, описания, картинки.
   - Поле для ссылки и шаблон ссылки (например, https://stalker-co.ru/stock/{value}/).
5. Сгенерируйте API-ключ и сохраните настройки.

Использование API:
------------------
URL: https://вашсайт.ru/bitrix/tools/mlk_tgbotapi_banner.php?key=КЛЮЧ&code=КОД_КОНТРАГЕНТА

Ответ:
------
{
  "contragent": {
    "code": "код",
    "name": "название",
    "segments": [1,2,3]
  },
  "banners": [
    {
      "id": 123,
      "iblock_id": 1,
      "name": "Баннер",
      "description": "Описание",
      "image": "https://...",
      "link": "https://stalker-co.ru/stock/example/",
      "date_from": "2024-01-01 00:00:00",
      "date_to": "2024-01-31 23:59:59",
      "segments": [1,2]
    }
  ]
}

Получение промо по одному или нескольким внешним кодам:
   - Один код:   /bitrix/tools/mlk_tgbotapi_promo.php?key=КЛЮЧ&promoid=CODE1
   - Несколько:  /bitrix/tools/mlk_tgbotapi_promo.php?key=КЛЮЧ&promoids[]=CODE1&promoids[]=CODE2
   - Строкой:    /bitrix/tools/mlk_tgbotapi_promo.php?key=КЛЮЧ&promoids=CODE1,CODE2

Ответ (всегда массив, даже для одного):
[
  {
    "id": 123,
    "iblock_id": 1,
    "name": "Акция 1",
    "image": "https://...",
    "link": "https://..."
  },
  ...
]
